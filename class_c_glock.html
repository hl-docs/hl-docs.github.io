Detailed Description

Represents the Glock 9mm handgun weapon in the game.

Represents the Glock pistol weapon in the game.

This class implements the functionality for the Glock pistol, including:

  • Primary and secondary fire modes
  • Weapon deployment and reloading
  • Ammo management
  • Sound and visual effects
  • Silencer attachment capability

This class implements the functionality for the Glock pistol, a semi-automatic handgun. It handles the weapon's firing mechanics, reloading, and animations.

Detailed Description

Defines the interface between the Half-Life engine and client DLL.

This header file contains the complete interface definition for communication between the Half-Life engine and client DLL. It includes:

  • Function pointer type definitions for client exports
  • Function pointer type definitions for engine exports
  • Structures for holding function pointers
  • Security module interface definitions
  • Engine data structures and helper functions

The file serves as a contract between the engine and client DLL, defining all the functions that can be called in either direction. It also includes versioning information and security-related structures for module validation.

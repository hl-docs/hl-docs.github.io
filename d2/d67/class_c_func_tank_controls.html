int CFuncTankControls::ObjectCaps ( void  )
virtual

Returns the object capabilities for the func_tankcontrols entity.

This function returns the object capabilities for the func_tankcontrols entity. It sets the FCAP_IMPULSE_USE capability and clears the FCAP_ACROSS_TRANSITION capability, meaning that the entity can be used with the USE command but cannot be moved across transitions.

Spawns the func_tank_controls entity.

This function initializes the func_tank_controls entity by setting its properties to be a trigger, not movable, not drawn, and with the specified model. It then sets the size and origin of the entity and schedules the first think with a delay of 0.3 seconds.

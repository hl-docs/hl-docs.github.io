edict_t * EntSelectSpawnPoint ( CBaseEntity *  pPlayer)
extern

Selects a spawn point for a player.

This function determines the appropriate spawn point for a player based on the game mode (Co-op or Deathmatch). It checks for valid spawn points and ensures that the selected point is not occupied by other players. If no valid spawn point is found, it defaults to a specified start point.

Parameters
pPlayerPointer to the player entity requesting the spawn.
Returns
Pointer to the selected spawn point entity.

Detailed Description

Implements a priority queue for pathfinding operations.

This class provides a priority queue implementation used for efficient pathfinding algorithms like A*. It maintains nodes in order of their priority (typically based on distance or cost) to help find optimal paths through the navigation graph.

Inserts a value into the priority queue.

This function adds a value to the priority queue with a specified priority. If the queue is full, it prints an error message and does not add the value. It also calls the Heap_SiftUp function to maintain the heap property.

Parameters

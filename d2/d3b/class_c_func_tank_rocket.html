virtual

Fires the RPG rocket entity.

This function is responsible for handling the firing of the RPG rocket entity. It checks if the tank has fired before and if so, calculates the number of rockets to fire based on the fire rate and the time elapsed since the last firing. It then creates the rockets and calls the base class's Fire() function.

Parameters
barrelEndThe end position of the tank's barrel.
forwardThe direction the tank is firing in.
pevAttackerThe entity that is attacking.

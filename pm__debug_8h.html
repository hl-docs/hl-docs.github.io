Shows the player's collision box and other debug information.

This function is used for debugging player movement and collision detection. It visualizes:

  1. The player's collision box
  2. The entity the player is looking at
  3. Various debug particles and lines

Note: This function only works in debug builds of the game.

void PM_ViewEntity ( void  )

Shows the entity currently in the player's view.

This function helps with debugging by:

  1. Drawing a line from the player to the entity they're looking at
  2. Showing the entity's bounding box
  3. Visualizing the intersection point where the player's view ray hits the entity

It's particularly useful for understanding collision detection and player interaction with objects.

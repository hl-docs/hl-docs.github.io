◆ Spawn()

void CDecal::Spawn ( void  )
virtual

Spawns the decal entity.

This function initializes the decal entity. If the skin is invalid or if the game is in deathmatch mode and the decal is not allowed, the entity is removed. If there is no target name, the decal will apply itself immediately after the world is done spawning. If there is a target name, the decal will apply itself when triggered.

void CDecal::StaticDecal ( void  )

Applies the decal statically.

This function applies the decal immediately at the specified location without waiting for a trigger. It performs a trace to determine where to apply the decal and then calls the engine function to create the static decal.

Triggers the decal to be applied.

This function is called when the decal entity is activated. It performs a trace to determine where to apply the decal and sends a message to all clients to display the decal at the specified location.

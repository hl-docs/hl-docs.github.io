◆ PM_CheckFalling()

void PM_CheckFalling ( void  )

Checks if the player is falling and handles the effects of a fall.

This function determines if the player is falling and handles the effects of a fall. The function first checks if the player is on the ground and if the player is not dead. If both conditions are met, the function checks if the player has fallen from a certain height and if so, plays a sound effect and knocks the screen around a little bit. The function also checks if the player has fallen from a height that is less than the safe fall height and if so, plays a sound effect.

The function then checks if the player is on the ground and if the player is still falling. If the player is still falling, the function sets the player's falling velocity to 0.

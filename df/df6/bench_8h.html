Performs special checks and setup for benchmarking entities.

This function is used during benchmarking (specifically in the third stage) to handle specific entities like "*3" or "ppdemodot". It calculates the aim origin (g_aimorg) and stores the origin of the dot entity (g_dotorg), possibly with some random vertical adjustment.

Sets the player's view angles during the benchmark.

Adjusts where the camera is looking during the benchmark based on the current benchmark stage. Also simulates aiming at a moving object by pointing the view direction toward g_aimorg.

Parameters
recalc_wanderWhether to recalculate random wandering (disabled in this version).
viewanglesThe view angles array to be modified.
frametimeTime passed since the last frame.
cmdUser command structure to modify input buttons (e.g., auto-firing).

Detailed Description

Defines the client DLL interface for Half-Life.

This header file contains the complete interface definition for the Half-Life client DLL. It includes:

  • Function declarations for client-side operations
  • Data structures for client state and rendering
  • Macros for engine function calls
  • Security module interface definitions
  • Client-side sprite and screen information structures

The file serves as the primary interface between the Half-Life engine and the client DLL, defining all the functions that the client DLL must implement and the structures it uses to communicate with the engine.

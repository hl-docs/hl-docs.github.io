int CHudMenu::MsgFunc_ShowMenu ( const char *  pszName,
int  iSize,
void *  pbuf 
)

Message handler for the ShowMenu message from the server.

This function handles incoming menu data, including:

  • Which keys the player can press
  • How long the menu should stay up
  • Whether more data is coming
  • The actual menu text to display

If no valid slots are given, the menu is hidden.

Parameters
pszNameThe message name (unused).
iSizeSize of the message in bytes.
pbufPointer to the message buffer.
Returns
Always returns 1.

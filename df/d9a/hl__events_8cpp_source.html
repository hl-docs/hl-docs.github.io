46/*
47* @brief Associate script file name with callback functions.
48*
49* Associate script file name with callback functions. Callback's must be extern "C" so
50* the engine doesn't get confused about name mangling stuff. Note that the format is
51* always the same. Of course, a clever mod team could actually embed parameters, behavior
52* into the actual .sc files and create a .sc file parser and hook their functionality through
53* that.. i.e., a scripting system.
54*
55* That was what we were going to do, but we ran out of time...oh well.
56*/

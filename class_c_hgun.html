Detailed Description

Represents the Hornet Gun weapon in the game.

This class implements the functionality for the Hornet Gun, a unique weapon that fires hornet entities. The weapon features both primary and secondary fire modes, with the primary mode firing single hornets and the secondary mode firing multiple hornets in a pattern. The weapon also includes ammo management, reloading mechanics, and visual/sound effects.

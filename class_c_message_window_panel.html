Constructs a message window panel with title and message content.

This constructor initializes a message window panel with a given MOTD (message of the day), title, and other properties. It sets up the background, title label, text content (in a scrollable panel), and a button to close or dismiss the message.

Parameters
szMOTDThe message content to be displayed in the window.
szTitleThe title of the message window.
iShadeFullscreenFlag to determine if the background should be shaded.
iRemoveMeFlag to determine whether the window should be removed.
xThe x-position of the window.
yThe y-position of the window.
wideThe width of the window.
tallThe height of the window.

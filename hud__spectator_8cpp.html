◆ V_GetChasePos()

void V_GetChasePos ( int  target,
float *  cl_angles,
float *  origin,
float *  angles 
)
extern

Calculates the camera position and orientation for a chase cam following a specific target.

This function determines the appropriate camera position and view angles based on the target entity's state and whether automatic camera direction is enabled. It will compute the camera position relative to the target, and can switch between different chase camera modes, such as the directed chase or a simple chase based on the player's angles.

Parameters
targetThe index of the target entity. If set to 0, a default position is used.
cl_anglesA pointer to the player's current view angles. If NULL, the target's angles are used.
originA pointer to the current position of the camera. It will be updated with the new camera position.
anglesA pointer to the camera's view angles. It will be updated based on the target's position and camera settings.

Sets the eye position and angles for a player or spectator.

This function calculates the position and angles based on the target entity, or sets a default position if no target is provided. It takes into account whether the player is dead, ducking, or in a normal state.

Parameters

◆ GetDirectorCamera()

bool CHudSpectator::GetDirectorCamera ( vec3_t &  position,
vec3_t &  angle 
)

Calculates the director camera view for the spectator.

Computes the camera's position and angle based on either a chased entity or a series of camera waypoints. If a chase target is valid, the camera is positioned relative to that entity. Otherwise, if waypoint interpolation is active, the camera's position is interpolated over time.

Parameters
positionOutput parameter that will contain the computed camera position.
angleOutput parameter that will contain the computed camera angle.
Returns
true if a valid camera view was computed; false otherwise.

◆ Eng_LoadFunctions()

int Eng_LoadFunctions ( HMODULE  hMod)

Loads the engine interface functions from the specified engine module (DLL).

This function attempts to locate and call the engine's API function, "Sys_EngineAPI", from a loaded engine DLL. If successful, it initializes the global engineapi structure with the function pointers required for further communication with the engine.

Parameters
hModHandle to the loaded engine module (DLL) from which the engine API will be retrieved.
Returns
int Returns 1 (true) if the engine functions were successfully loaded, 0 (false) otherwise.

◆ LoadSoundAPIs()

void LoadSoundAPIs ( void  )

Loads the appropriate engine DLL (hardware or software) and initializes sound API pointers.

This function dynamically loads either the hardware (hw.dll) or software (sw.dll) engine module based on whether the engine is running in hardware mode. After loading, it retrieves the DirectSound and waveform output pointers via the engine API if available.

This is used to initialize the sound subsystem for further use.

◆ ShutdownSoundAPIs()

void ShutdownSoundAPIs ( void  )

Shuts down and cleans up the sound APIs.

This function unloads the engine DLL if it was loaded, and resets all sound-related pointers (DirectSound interface, primary buffer, and waveform output) to null. It is typically called during shutdown or cleanup to release resources.

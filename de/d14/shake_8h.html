Detailed Description

Defines screen effects and view manipulation for the Half-Life engine.

This header file contains definitions for screen effects that enhance gameplay feedback and immersion. It includes:

  • Screen shake effects for impacts and explosions
  • Screen fade effects for transitions and damage
  • View manipulation functions
  • Network message handling for effects

These effects are used throughout the game to provide visual feedback for various events like explosions, damage, and level transitions.

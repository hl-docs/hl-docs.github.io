Detailed Description

Code for skill level concerns in the game.

This file contains the implementation of functionality related to skill levels in the game. It defines the gSkillData structure, which holds various parameters that affect gameplay based on the current skill level. The skill level can influence monster health, damage, and player weapon effectiveness, allowing for a dynamic difficulty adjustment.

The global variable gSkillData is an instance of the skilldata_t structure, which includes fields for health and damage values for various monsters, as well as player weapon damage values. This allows the game to scale difficulty based on the selected skill level.

The primary function in this file is GetSkillCvar, which retrieves the value of a console variable (CVAR) based on the current skill level. This function constructs a variable name by appending the skill level to a base name and retrieves the corresponding value from the engine's configuration variables.

The function performs the following steps:

  • Constructs a variable name by appending the current skill level (stored in gSkillData.iSkillLevel) to the base name provided in the parameter.
  • Retrieves the float value associated with the constructed variable name using the CVAR_GET_FLOAT function.
  • If the retrieved value is less than or equal to zero, it prints a warning message to the console indicating that a zero value was obtained for the constructed variable name.

float GetSkillCvar ( char *  pName)

Retrieves the skill variable value based on the current skill level.

This function constructs a variable name based on the provided name and the current skill level, then retrieves the corresponding value from the engine's configuration variables (CVAR). It is used to adjust game difficulty by accessing skill-related parameters defined in the engine.

Parameters
pNameA pointer to a character string that represents the base name of the skill variable. This name will be combined with the current skill level to form the complete variable name.
Returns
Returns the float value of the skill variable. If the retrieved value is less than or equal to zero, a warning message is printed to the console indicating that a zero value was obtained for the constructed variable name.

The function performs the following steps:

  • It constructs a variable name by appending the current skill level (stored in gSkillData.iSkillLevel) to the base name provided in pName. The resulting name is stored in a buffer (szBuffer).
  • It retrieves the float value associated with the constructed variable name using the CVAR_GET_FLOAT function.
  • If the retrieved value is less than or equal to zero, it prints a warning message to the console using the ALERT function, indicating that a zero value was obtained for the variable.

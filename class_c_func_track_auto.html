void CFuncTrackAuto::UpdateAutoTargets ( int  toggleState)
virtual

Updates the auto targets for the track change based on the current state.

This function enables or disables the path tracks based on whether the track change is at the top or bottom position. It also activates the train if it is following the track and is currently stopped.

Parameters
toggleStateThe current state of the toggle (top or bottom).

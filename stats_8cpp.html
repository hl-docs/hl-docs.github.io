void UpdateStats ( CBasePlayer *  pPlayer)

Updates the game statistics for a given player.

This function collects and updates various statistics related to the player's health, ammunition, and game time. It calculates the total damage potential of the player's ammunition and checks for significant changes in health and ammo to determine if the statistics should be written to the stats file.

Parameters
pPlayerA pointer to the CBasePlayer object representing the player whose statistics are being updated.

The function performs the following actions:

  • Copies the player's current ammunition counts into a local array.
  • Updates the game time, ensuring it does not overlap if the level has changed or the player has died.
  • Iterates through the player's items to calculate the total ammo available and its corresponding damage.
  • Calculates the player's total health, factoring in armor.
  • Compares the current ammo and health values to the last recorded values to determine if significant changes have occurred.
  • If changes exceed predefined thresholds, it schedules an update to the stats file.

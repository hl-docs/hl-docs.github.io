void CXenTreeTrigger::Touch ( CBaseEntity *  pOther)
virtual

Handles the touch event for the Xen tree trigger.

This function is called when another entity touches the trigger. It forwards the touch event to the owner of the trigger if it exists.

Parameters
pOtherPointer to the entity that touched the trigger.

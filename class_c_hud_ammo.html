void CHudAmmo::UserCmd_NextWeapon ( void  )

Selects the next weapon in the inventory, ensuring it has available ammo.

This function checks the current selected weapon and attempts to cycle through the available weapons in the inventory. If no weapon is selected, it defaults to the current weapon. The function will skip any weapons without ammo. The selection happens by iterating through the slots and positions for weapons.

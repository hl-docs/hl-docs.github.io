Writes data to the demo stream.

This function takes a type identifier, a size, and a buffer containing data to be written to the demo stream. It packs the data and sends it through the gEngfuncs.pDemoAPI->WriteBuffer method to write the buffer.

Parameters
typeThe type of the data being written (e.g., buffer type identifier).
sizeThe size of the data in the buffer.
bufferA pointer to the data buffer to be written.

Detailed Description

Represents a node entity in the game world.

This class is used to place navigation nodes in the level during map creation. These entities are used to build the navigation graph and are then removed when the game starts. They can include hints for AI behavior at specific locations.

Handles key-value pairs for the node entity.

This function processes key-value pairs for the node entity. It sets the hint type and activity based on the provided key-value data. If the key is not recognized, it calls the base class's KeyValue function.

void CNodeEnt::Spawn ( void  )
privatevirtual

Spawns the node entity.

This function initializes the node entity's properties and determines its behavior based on the state of the node graph. If the graph is loaded, the node entity is removed immediately. If this is the first node to spawn, it also spawns the test hull entity that builds and walks the node tree.

Detailed Description

Represents the Rocket Propelled Grenade (RPG) weapon in the game.

This class implements the functionality for the RPG, a heavy weapon that fires guided rockets. It manages the weapon's laser targeting system, rocket guidance, and explosion effects. The RPG features a laser sight for improved accuracy and can track targets with its guided rockets.

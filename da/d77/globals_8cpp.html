Detailed Description

DLL-wide global variable definitions.

This file contains the definitions of global variables that are used throughout the DLL. These variables are declared with the DLL_GLOBAL macro to ensure they are accessible across different source files within the DLL. The purpose of centralizing these global variables is to provide a single point of reference for their definitions, making it easier to manage and document their usage.

The global variables defined in this file include:

  • g_ulFrameCount: A counter for the number of frames processed.
  • g_ulModelIndexEyes: The model index for the player's eyes.
  • g_ulModelIndexPlayer: The model index for the player character.
  • g_vecAttackDir: A vector representing the direction of an attack.
  • g_iSkillLevel: An integer representing the current skill level of the game.
  • gDisplayTitle: An integer used to control the display of the title.
  • g_fGameOver: A boolean flag indicating whether the game is over.
  • g_vecZero: A constant vector representing a zero vector (0,0,0).
  • g_Language: An integer representing the current language setting.

Source files that require access to these global variables should declare them as extern to document their dependencies and ensure proper linkage.

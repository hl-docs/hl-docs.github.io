◆ Trace_GetHopCount()

int Trace_GetHopCount ( char *  pServer,
int  nMaxHops 
)

Performs a synchronous traceroute to determine the number of hops to a specified server.

Uses Windows ICMP APIs (via ICMP.DLL) to send echo requests with increasing TTL values. Returns when the destination is reached or the max hops limit is exceeded.

Parameters
pServerHostname or IP address of the target server.
nMaxHopsMaximum number of hops to probe.
Returns
Number of hops if successful, or -1 on failure (DLL load failure, DNS failure, or timeout).

void CItem::ItemTouch ( CBaseEntity *  pOther)

Handles the touch event for the item.

This function is called when a player touches the item. It checks if the entity touching the item is a player and whether the player can pick up the item. If the player can pick it up, the item is given to the player, and the item is either removed or set to respawn.

Parameters
pOtherPointer to the entity that touched the item.

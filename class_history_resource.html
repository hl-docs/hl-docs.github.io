Adds an entry to the ammo history based on sprite name.

This function adds an item (like a weapon or item) to the ammo history, recording the sprite name, type, and count. It ensures that the sprite name is valid before adding it to the history. The history entry's display time is set based on a configurable value.

Parameters
iTypeThe type of the history entry (e.g., item or weapon).
szNameThe sprite name associated with the item or weapon.
iCountThe count of the item or weapon picked up.

Adds an entry to the ammo history.

This function adds an ammo pickup to the history, recording the ammo type, ID, and count. It also handles managing the history slots, ensuring that history items are added in the correct slot and that items are drawn within the allowed height range. The history entry's display time is set based on a configurable value.

Parameters
iTypeThe type of the history entry (e.g., ammo type).
iIdThe ID of the ammo or weapon.
iCountThe count of the ammo or weapon picked up.

Clears the ammo history if all entries are empty.

This function checks the ammo history slots to see if all of them are empty (no entries). If all slots are empty, it resets the history slot counter to zero.

int HistoryResource::DrawAmmoHistory ( float  flTime)

Draws the ammo pickup history on the HUD.

This function draws the history of ammo pickups (or item pickups) on the HUD. It displays either ammo icons, weapon icons, or item sprites, with a countdown for how long they should remain visible. Once the history item's display time expires, it is removed from the history. It scales the colors based on the time left before the item disappears.

Parameters
flTimeThe current time, used to check the expiry of history items.
Returns
int Returns 1 after drawing the history items.

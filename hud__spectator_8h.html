void VectorAngles ( const float *  forward,
float *  angles 
)
extern

Converts a forward direction vector into a corresponding yaw and pitch (angles).

This function converts a 3D forward direction vector into its corresponding yaw and pitch angles. The yaw is the horizontal rotation angle around the vertical axis, while pitch is the vertical rotation angle around the horizontal axis. The result is stored in the angles array.

Parameters
forwardThe input forward vector to calculate angles from.
anglesThe output angles array to store the resulting pitch (angles[0]) and yaw (angles[1]).

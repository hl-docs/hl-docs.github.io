Detailed Description

Represents a wall-mounted health charger in the game.

This class implements a stationary health charger that players can use to restore their health. It features continuous use capability, recharge mechanics, and sound effects for operation.

void CWallHealth::KeyValue ( KeyValueData *  pkvd)
virtual

Processes key-value pairs for the wall health entity.

This function handles specific key-value pairs related to the wall health settings. It sets the appropriate member variables based on the provided key names.

Parameters
pkvdPointer to a KeyValueData structure containing the key-value pair to be processed.

The function handles the following keys:

  • "style", "height", "value1", "value2", "value3": These keys are recognized but do not affect the wall health entity's behavior.
  • "dmdelay": Sets the reactivation delay for the wall health entity.

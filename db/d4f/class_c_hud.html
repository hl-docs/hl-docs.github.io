USER-DEFINED SERVER MESSAGE HANDLERS.

Resets the HUD data and other related variables.

This function is called when a reset event is triggered (e.g., after a level reset or player respawn). It clears all the HUD data by iterating through the list of HUD elements and calling their Reset method. It also resets some game-related variables such as mouse sensitivity and concussion effect.

Parameters
pszNameThe name of the message (not used in this function).
iSizeThe size of the message buffer (expected to be 0).
pbufA pointer to the message buffer (not used in this function).
Returns
Returns 1 to indicate success.

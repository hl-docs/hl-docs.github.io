void CTestHull::BuildNodeGraph ( void  )

Builds the node graph by linking visible nodes.

This function is called by the test hull to establish connections between nodes that can see each other. It creates a temporary link pool, checks for visibility between nodes, and eliminates any inline links that do not provide valid paths. The function also generates a report of the node graph and saves it to a file.

It performs the following steps:

  • Initializes the temporary link pool.
  • Creates directories for saving the node graph report.
  • Iterates through all nodes to determine their types (AIR, WATER, LAND).
  • Links visible nodes and checks if they can be traversed by different hull sizes.
  • Validates link pairings to ensure that connections are mutual.

Detailed Description

Base class for all entities in the game.

This is the fundamental class from which all game entities derive. It provides common functionality for entity management, including spawning, saving/restoring, collision detection, and basic entity interactions. The class handles core entity behaviors and provides virtual methods that derived classes can override to implement specific functionality.

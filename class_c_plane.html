Detailed Description

Represents a mathematical plane in 3D space.

The CPlane class is used to define a plane in three-dimensional space using a normal vector and a point on the plane. It provides methods to initialize the plane with these parameters and to check if a given point is in front of the plane. The plane is defined by the equation: 

dot(normal, point) = distance

where 'normal' is the normal vector of the plane, 'point' is a point on the plane, and 'distance' is the distance from the origin to the plane.

This class is essential for collision detection and geometric calculations in 3D environments.

Initializes the plane with a normal vector and a point.

This function sets the normal vector of the plane and calculates the distance from the origin to the plane using the provided point. It also marks the plane as initialized.

Parameters
vecNormalA vector representing the normal of the plane. This vector is perpendicular to the surface of the plane.
vecPointA point on the plane used to calculate the distance from the origin to the plane.

The distance from the origin to the plane is calculated using the dot product of the normal vector and the point on the plane. This distance is stored for later use in determining the position of points relative to the plane.

BOOL CPlane::PointInFront ( const Vector &  vecPoint)

Checks if a point is in front of the plane.

This function determines whether the specified point is located in front of the plane defined by its normal vector and distance. A point is considered to be in front of the plane if the dot product of the normal vector and the point minus the distance is greater than or equal to zero.

Parameters
vecPointThe point to check against the plane.
Returns
Returns TRUE if the point is in front of the plane, and FALSE if the plane has not been initialized or the point is behind the plane.

The function first checks if the plane has been initialized. If it has not, it returns FALSE. If it has been initialized, it calculates the value of flFace using the dot product and compares it to the distance to determine the position of the point relative to the plane.

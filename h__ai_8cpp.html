Detailed Description

Half-Life specific AI code.

This file contains the implementation of various AI utility functions specific to the Half-Life game. These functions are designed to assist in the behavior and decision-making processes of monsters within the game.

Key functionalities provided in this file include:

  • Visibility checks: Functions to determine if a target is visible from a monster's perspective, taking into account water levels and line of sight.
  • Toss and throw calculations: Functions to calculate the necessary velocity for tossing or throwing objects from one point to another, considering gravity and obstacles.

The utility functions in this file are essential for enabling dynamic and responsive AI behavior, allowing monsters to interact with their environment effectively.

Checks if the target is visible from the looker's perspective.

This function determines if the looker can see the target based on their positions and water levels. It performs a series of trace checks to establish a line of sight.

Parameters
pevLookerPointer to the entity variables of the looker.
pevTargetPointer to the entity variables of the target.
vecTargetOriginReference to a vector that will store the target's position if visible.
flSizeThe size offset to apply when checking visibility.
Returns
Returns TRUE if the target is visible, FALSE otherwise.

Calculates the velocity needed to throw an object from one point to another.

This function determines the velocity vector at which an object should be thrown from vecSpot1 to hit vecSpot2. It returns a zero vector if the throw is not feasible.

Parameters

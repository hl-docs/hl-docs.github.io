Provides the studio renderer interface to the engine.

This function exports the studio renderer interface so the engine can use it to render studio models. The function copies engine helper functions into the provided interface and initializes necessary variables and resources.

Parameters
versionThe version of the interface being requested. It must match the STUDIO_INTERFACE_VERSION.
ppinterfaceA pointer to a pointer that will receive the studio interface.
pstudioA pointer to the engine's studio API, which is copied into the interface.
Returns
1 on success, 0 if the version mismatch occurs.

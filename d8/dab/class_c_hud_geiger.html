int CHudGeiger::Draw ( float  flTime)
virtual

Draws the Geiger counter HUD element on the screen and handles its sound effects.

This function checks the current Geiger range and uses a piecewise linear approach to determine the Geiger counter's state. It then generates a random volume for the sound effect and plays a sound depending on the Geiger range. The sound is played at varying volumes and frequencies based on the current Geiger range.

Parameters
flTimeThe current time, which may be used for animation or sound timing.
Returns
Returns 1 to indicate successful drawing of the Geiger counter HUD element.

int CHudGeiger::Init ( void  )

Initializes the Geiger counter HUD element.

This function initializes the Geiger counter HUD element by hooking the Geiger message, setting the initial Geiger range to 0, and setting the initial HUD flags. It also adds the Geiger counter element to the HUD list and seeds the random number generator for later use in generating sound effects.

Returns 1 to indicate successful initialization.

Handles the incoming Geiger counter message and updates its range.

This function processes the Geiger counter message, which contains the current Geiger range. The range is then shifted and used to update the internal m_iGeigerRange value, which will influence the sound effects and behavior of the Geiger counter.

Parameters
pszNameThe name of the message (not used in this function).
iSizeThe size of the message buffer.
pbufA pointer to the message buffer containing the Geiger counter range data.
Returns
Returns 1 to indicate successful processing of the message.

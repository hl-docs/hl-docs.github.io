Adds an entity to the full update pack for the client.

This function checks if the entity state has been filled in for the entity and determines if it should be propagated to the client. It also allows for modifications to the state before sending it to the client.

Parameters
statePointer to the server-maintained copy of the entity state.
eThe entity index.
entPointer to the entity being added to the update.
hostPointer to the player's edict who is receiving the update.
hostflagsFlags associated with the host.
playerIndicates if the entity is a player (1) or not (0).
pSetPointer to the PAS or PVS set up previously.
Returns
1 if the entity state has been filled in and will be sent to the client, 0 otherwise.

◆ GetVGUITGAName()

char * GetVGUITGAName ( const char *  pszName)

Gets the name of the TGA file based on the game's directory and screen width.

This function takes a file name and changes its name based on the current screen resolution. If the screen width is smaller than 640, it changes the file name to include a 320px variant. If the screen width is greater than or equal to 640, it uses a 640px variant.

Parameters
pszNameThe base name of the TGA file.
Returns
A pointer to a string containing the full path of the TGA file.

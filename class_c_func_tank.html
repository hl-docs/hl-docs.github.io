Shoots a trace in a random direction to simulate the tank's innaccuracy.

This function creates a random direction based on a circular gaussian distribution and traces a line in that direction. The random direction is done by adding two random numbers between -0.5 and 0.5 together, and then rejecting the result if the magnitude of the resulting vector is greater than 1.0.

Parameters
vecStartThe starting position of the trace.
vecForwardThe direction the tank is pointing.
vecSpreadThe spread of the shot (in degrees).
trThe trace result structure to fill in.

void CFuncTank::Think ( void  )
virtual

The think function for the func_tank entity.

This function is called once a frame and is responsible for rotating the tank to face its target. It first zeroes the angular velocity of the tank, then calls TrackTarget to calculate the angular velocity needed to face the target. Finally, it checks if the angular velocity is greater than 1 and if so, starts the rotation sound, otherwise it stops the sound.

◆ TrackTarget()

void CFuncTank::TrackTarget ( void  )

Tracks and aims the tank at a target.

This function calculates and adjusts the tank's aiming angles toward a target. If a controller exists, the tank aims based on the player's view angles. Otherwise, it attempts to find and track a player in its view. The function checks for line of sight and range before adjusting the aim and potentially firing at the target.

If the target is out of range or line of sight is obstructed, the tank will not track or fire. The function also handles barrel adjustments and limits the rotation rates of the tank.

void CFuncTank::Use ( CBaseEntity *  pActivator,

USE_TYPE  useType,
float  value 
)
virtual

Handles the use action of the func_tank entity.

This function is called when another entity uses the func_tank entity. It checks if the entity is a player-controlled turret and if so, allows the player to control the turret. The player can start controlling the turret by calling the Use function on the turret with the USE_ON value, and can stop controlling the turret by calling the Use function with the USE_OFF value. The player can also fire the turret by calling the Use function with the USE_SET value and the value of 2.

If the entity is not a player-controlled turret, the function checks if the entity should toggle its state between on and off based on the type of use action. If the entity should toggle, it turns itself on or off depending on its current state.

Parameters
pActivatorPointer to the entity that activated the use.
pCallerPointer to the entity that called the use.
useTypeThe type of use action (e.g., ON, OFF).
valueAdditional value for the use action.

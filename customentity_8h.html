Detailed Description

Defines constants and structures for custom entity types in Half-Life.

This header file contains definitions for custom entity types, particularly focusing on beam effects and their properties. It includes:

  • Beam entity encoding macros
  • Beam type enumerations
  • Beam effect flags and properties

Detailed Description

Defines program state and entity management functions for the Half-Life engine.

This header file contains definitions for managing program state and entities in the Half-Life engine. It includes:

  • Event queue and state management
  • Entity allocation and management functions
  • Program string and global variable access
  • Entity dictionary utilities

This file provides the core functionality for managing the engine's program state, including event handling, entity lifecycle management, and program data access.

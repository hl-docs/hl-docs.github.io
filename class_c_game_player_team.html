virtual

Activates the game player team entity.

This function is called when the game player team entity is used by another entity. It checks if the activator can trigger the action and changes the player's team based on the target team name.

Parameters
pActivatorPointer to the entity that activated the game player team.
pCallerPointer to the entity that called the game player team.
useTypeThe type of use action (e.g., ON, OFF).
valueAdditional value for the use action.

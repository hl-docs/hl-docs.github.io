Detailed Description

Represents a navigation node in the game world.

This class defines a point in space that can be used for AI navigation. Each node contains information about its position, connections to other nodes, and various properties that affect how entities can move through it. Nodes are used to create a navigation network that AI entities can use to move around the game world.

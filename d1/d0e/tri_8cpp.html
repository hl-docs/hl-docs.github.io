void CL_DLLEXPORT HUD_DrawNormalTriangles ( void  )

Draws non-transparent (opaque) triangles on the HUD.

This function is used to render all solid geometry that is part of the player's heads-up display (HUD). It is mainly used for spectator mode where an overhead overview map may be shown.

Called every frame after opaque world and entity rendering is done, but before transparent elements are drawn.

Note
You can place any custom opaque triangle rendering code here.

void CL_DLLEXPORT HUD_DrawTransparentTriangles ( void  )

Draws transparent triangles on the HUD.

This function is called after all opaque geometry is drawn. It is used for drawing transparent effects, such as particles or overlays that need blending (like smoke, laser beams, etc.).

If running in a mod like Team Fortress Classic (TFC), it may process additional event effects.

Note
Particle systems are updated here if a particle manager is available.

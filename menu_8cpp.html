int KB_ConvertString ( char *  in,
char **  ppout 
)

Replaces any "+command" strings in the input with their corresponding key bindings.

This function scans the input string for segments starting with '+' (e.g., "+use") and attempts to find the key binding for that command using Key_LookupBinding(). If found, it replaces the command with the key name in square brackets. If not found, it leaves the original text unchanged.

Parameters
inThe input string containing +commands (e.g., "+use").
ppoutPointer to a char pointer that will be allocated and filled with the converted output string.
Returns
Returns 1 if successful, 0 if ppout is null.

Parses a formatting escape token in the menu string.

These escape tokens change the color or alignment of the text:

  • \w : White text
  • \d : Dim (gray) text
  • \y : Yellow text
  • \r : Red text
  • \R : Right-align text on the current line
Parameters
tokenA string beginning with a backslash, followed by a character code.
Returns
Pointer to the next character after the escape code.

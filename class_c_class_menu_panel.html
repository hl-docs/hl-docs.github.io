Constructor for the class menu panel.

This creates and initializes the class selection menu that appears when a player is choosing a class. It sets up the GUI layout, labels, buttons, and loads images and text descriptions for each class.

Parameters
iTransTransparency value for the menu background.
iRemoveMeA flag to determine if the panel should be removed after use.
xThe x-position of the panel.
yThe y-position of the panel.
wideWidth of the panel.
tallHeight of the panel.

When the player hovers over or selects a class button, this function shows the class description, image, and name related to that class. It also hides all other class info panels.

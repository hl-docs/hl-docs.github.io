Detailed Description

Represents the player entity in the game.

This class implements the core functionality for player characters in Half-Life. It handles player movement, combat, inventory management, and various game mechanics specific to player characters. The class extends CBaseMonster to inherit basic entity functionality while adding player-specific features like:

  • Weapon and inventory management
  • Health and armor systems
  • Movement and physics
  • HUD and UI interactions
  • Observer mode and camera controls
  • Suit power and environmental effects
  • Team and multiplayer functionality

Vector VecBModelOrigin ( entvars_t *  pevBModel)
extern

Calculates the origin of a bounding model.

This function computes the origin point of a bounding model based on its minimum and size properties. The origin is determined as the center of the bounding box defined by the model's dimensions.

Parameters
pevBModelPointer to the entity's variables containing bounding model data.
Returns
The calculated origin as a Vector.

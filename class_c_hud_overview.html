int CHudOverview::Draw ( float  flTime)

Renders the overview map and player markers.

This is the main rendering function for the overview mode. Draws all active players on the overview map using team-specific colors, health display, and sprite markers.

Parameters
flTimeCurrent client time (unused).
Returns
1 if successful or if not drawing due to inactive overview mode.

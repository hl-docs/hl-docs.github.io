Draws the ammo bars for primary and secondary ammo types.

This function calculates and renders two ammo bars (one for primary ammo and another for secondary ammo), indicating the amount of ammo the player has. The bars are drawn horizontally and are filled proportionally to the current ammo.

Parameters
pThe weapon whose ammo bars need to be drawn.
xThe x position to start drawing the ammo bars.
yThe y position to start drawing the ammo bars.
widthThe width of the ammo bars.
heightThe height of the ammo bars.

Draws an individual ammo bar on the HUD.

This function renders a colored bar representing the available ammo for a specific weapon. The bar is filled proportionally to the current ammo count, and the color indicates the ammo level (e.g., green for full, yellow for medium, etc.).

Parameters
xThe x position to start drawing the bar.
yThe y position to start drawing the bar.
widthThe width of the ammo bar.
heightThe height of the ammo bar.
fA float representing the ammo fraction (0.0 to 1.0).
Returns
The x-coordinate of the end of the ammo bar.

Searches for a sprite by name and resolution in a sprite list.

This function iterates through a list of sprites to find one that matches the specified name and resolution. If a match is found, a pointer to the corresponding client_sprite_t structure is returned.

Parameters
pListPointer to the array of client_sprite_t structures (the sprite list).
pszName of the sprite to search for.
iResDesired resolution of the sprite (usually corresponding to screen resolution).
iCountNumber of elements in the sprite list.
Returns
Pointer to the matching client_sprite_t if found; otherwise, NULL.

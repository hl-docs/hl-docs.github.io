Selects the appropriate weapon slot and manages weapon selection.

This function is used to select a weapon slot for either advancing or switching to the next weapon within the specified slot. It manages the fast-switch behavior and displays the weapon list if required. It also handles the case where only one weapon is available in the current slot.

Parameters
iSlotThe index of the slot to select.
fAdvanceA flag indicating if the selection is for advancing.
iDirectionThe direction in which to advance.

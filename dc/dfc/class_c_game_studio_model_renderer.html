void CGameStudioModelRenderer::StudioSetupBones ( void  )
virtual

Sets up the bones for rendering the current studio model.

This method handles bone setup differently for player models versus non-player models. For player models, it supports three-way animation blending and transitions between animation sequences.

Handles:

  • Standard bone rotation calculation
  • Three-way blending (Left-Mid-Right)
  • Interpolated transitions between sequences
  • Quaternion to matrix conversion
  • Matrix transformation for rendering and lighting

Creates and initializes a message window panel.

This function creates a new message window panel, which is used to display a message (like MOTD - Message of the Day) in a window with a title. It sets the position, size, and other properties of the window, and prepares the components (like the title, message text, scroll panel, and buttons).

Parameters
szMOTDThe message of the day (the main content to be displayed).
szTitleThe title of the message window.
iShadeFullscreenIf true, the window has a shaded background; otherwise, no shading.
iRemoveMeA flag that indicates if the window should be removed.
xThe x-position of the window.
yThe y-position of the window.
wideThe width of the window.
tallThe height of the window.
Returns
CMenuPanel* Returns a pointer to the created menu panel.

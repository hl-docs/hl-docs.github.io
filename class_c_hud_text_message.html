Localizes a text message by replacing any placeholder messages with actual text.

This function searches the input string for any placeholders marked by '#' characters. Each placeholder is replaced with the corresponding text from the titles.txt file. The result is written into the destination buffer.

Parameters

dst_bufferThe destination buffer to store the localized message.
buffer_sizeThe size of the destination buffer.
Returns
A pointer to the destination buffer containing the localized message.

char * CHudTextMessage::LookupString ( const char *  msg,
int *  msg_dest = NULL 
)

Simplified version of LocaliseTextString, assumes the message is a single word.

This function checks if the input message is a reference to another message in titles.txt. If the message starts with '#', it will be treated as a message name and replaced with the corresponding message from titles.txt. If no such message is found, it returns the original string.

Parameters
msgThe input message string.
msg_destOptional parameter to store the message destination if it is defined in titles.txt.
Returns
A pointer to the message string (either the original or the localized one).

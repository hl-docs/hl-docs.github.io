Returns the current state of a key as a float value.

Used to determine how long a key was pressed during the frame:

  • 1.0: Held the entire frame
  • 0.5: Pressed and held this frame

Handles a key down event for a specific key binding (kbutton_t).

Called when a key is pressed. Updates the internal state to track which keys are holding down this binding. Also marks the button as "impulse down", which means it was just pressed this frame.

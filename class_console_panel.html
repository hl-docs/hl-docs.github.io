Constructs a ConsolePanel with specified position and size.

This panel includes a text grid for console output and a text entry for user input. It also sets appropriate borders and parenting for the UI elements.

Parameters
xThe x-coordinate of the panel.
yThe y-coordinate of the panel.
wideThe width of the panel.

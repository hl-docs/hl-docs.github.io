Detailed Description

Manages the complete navigation graph for the game world.

This class is responsible for maintaining and managing the entire navigation system. It handles node creation, pathfinding, and routing information. The graph is used by AI entities to determine how to move through the game world efficiently.

◆ InitGraph()

void CGraph::InitGraph ( void  )

Initializes the graph by freeing allocated resources and resetting parameters.

This function is responsible for preparing the graph for use by:

  • Marking the graph as unavailable.
  • Freeing any memory currently allocated for nodes, links, and other related structures.
  • Setting all pointers to NULL to avoid dangling references.
  • Resetting the node and link counts to zero.
  • Resetting the last search indices to ensure a fresh start for future operations.

This function should be called before using the graph to ensure that it is in a clean state.

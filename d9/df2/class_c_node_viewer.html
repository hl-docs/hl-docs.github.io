void CNodeViewer::Spawn ( void  )
virtual

Initializes the node viewer entity.

This function checks if the node graph is present and properly set up. It determines the hull type and color based on the class name of the entity. It finds the nearest node to the viewer's position and initializes connections to other nodes based on the graph.

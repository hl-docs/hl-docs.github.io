float GetSkillCvar ( char *  pName)

Retrieves the skill variable value based on the current skill level.

This function constructs a variable name based on the provided name and the current skill level, then retrieves the corresponding value from the engine's configuration variables (CVAR). It is used to adjust game difficulty by accessing skill-related parameters defined in the engine.

Parameters
pNameA pointer to a character string that represents the base name of the skill variable. This name will be combined with the current skill level to form the complete variable name.
Returns
Returns the float value of the skill variable. If the retrieved value is less than or equal to zero, a warning message is printed to the console indicating that a zero value was obtained for the constructed variable name.

The function performs the following steps:

  • It constructs a variable name by appending the current skill level (stored in gSkillData.iSkillLevel) to the base name provided in pName. The resulting name is stored in a buffer (szBuffer).
  • It retrieves the float value associated with the constructed variable name using the CVAR_GET_FLOAT function.
  • If the retrieved value is less than or equal to zero, it prints a warning message to the console using the ALERT function, indicating that a zero value was obtained for the variable.

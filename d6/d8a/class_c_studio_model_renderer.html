void CStudioModelRenderer::StudioMergeBones ( model_t *  m_pSubModel)
virtual

Merges bone transformations from a sub-model into the current model.

This function performs a merge of bone transformations between the current model and a sub-model. It checks if bones from the sub-model exist in the cached bone names and applies transformations accordingly. If a bone is not cached, it computes the transformation matrix using the bone's position and quaternion rotation.

Parameters
m_pSubModelA pointer to the sub-model whose bone transformations will be merged.

Processes the player's gait and adjusts the torso and animation parameters.

This function calculates and adjusts the player's gait frame, yaw, and torso position based on the player's movement and animation. The gait frame determines the frame in the animation sequence based on the player's movement speed. The torso is adjusted to match the player's turning direction.

Parameters
pplayerThe current player entity, containing the player's state and animation sequence information.

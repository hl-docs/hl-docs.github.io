void ScorePanel::HitTestPanel::internalMousePressed ( MouseCode  code)
virtual

Handles mouse press events inside the HitTestPanel.

This function loops through all input signals (objects that handle input events) and tells them that the mouse was pressed. It's a way of passing the event to all parts of the UI that might care about mouse input on this panel.

Parameters
